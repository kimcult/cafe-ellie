# Cafe Ellie — Site Vision

## 1. Project Overview
**Brand:** Cafe Ellie
**Slogan:** "커피 한 잔의 진심"
**Concept:** Modern Korean specialty cafe with Apple-inspired minimal web presence. Emotion-driven design, photography-forward, one message per screen.

## 2. Target Audience
Urban coffee enthusiasts, 25–40. Design-conscious. Value quality, atmosphere, and authenticity over noise.

## 3. Technical Stack
- Pure HTML/CSS/JS (no framework)
- Scroll animations: Intersection Observer API for fade-in
- Parallax: CSS transform on scroll
- Smooth page transitions
- Google Fonts or system font stack
